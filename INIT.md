# INIT：为什么发起“问题实验室”

我们每天都会遇到很多问题。

有些问题看似是选择问题：

- 我要不要继续做？
- 我要不要离开？
- 我要不要开始？
- 我要不要回应别人期待？

有些问题看似是能力问题：

- 我是不是不适合？
- 我是不是做不到？
- 我是不是又失败了？

有些问题看似是关系问题：

- 如果我不帮，会不会显得冷漠？
- 如果我不兜底，事情会不会崩？
- 如果我不满足别人期待，是不是我不好？

但很多时候，真正困住我们的，不一定是问题本身，而是我们提问的方式。

这个项目想做的，是和大家一起探索：

> 如何把一个模糊、情绪化、二元化的问题，改写成一个可以观察、可以实验、可以行动的问题。

思想实验是我们的工具。

它不是为了给出标准答案，而是为了帮助我们：

- 换一个位置看问题；
- 换一个时间尺度看问题；
- 换一种语言命名问题；
- 换一个系统视角理解问题；
- 把人生定论改成行动实验。

## 项目的初始假设

我们暂时认为：

1. 人的问题并不总是需要被直接回答，很多问题需要先被重新组织。
2. 一个好的思想实验，可以改变问题的变量、视角、时间尺度或情境边界。
3. 认知上的“有启发”还不够，新的理解需要通过行动实验接受现实反馈。
4. 一个思想实验是否有效，需要在真实案例中被检验。
5. 社群可以共同沉淀思想实验的适用情境、失效边界和行动结果。

## 这个项目不是一次活动

它不是一次讲座，不是一次咨询，也不是一次打卡。

它是一个持续生长的共创库。

每一次活动，我们会围绕一个思想实验展开；每一个参与者，可以带来一个真实问题；每一次使用之后，我们都会记录：

- 它有没有帮助；
- 它帮助了什么；
- 它在哪里失效；
- 它带来了什么行动实验；
- 现实反馈如何修正了我们的理解。

我们希望长期积累一套来自真实生活的思想实验使用手册。
