# 认知方式（Ways of Knowing）

> 源材料来自阳志平《聪明的阅读者》第六章「结构阅读：掌握作者的认知方式」
> 整理时间：2026-07-03
> 用途：知识库原始资料，可作为后续产品设计、内容创作的底层框架

---

## 一、什么是认知方式

**认知方式**（ways of knowing）是指个人思维、知觉、记忆以及运用知识解决问题的方式，近义词包括认识方式、认知框架等。

哈佛大学心理学家戴维·珀金斯认为，各学科采用的具体论证方法虽各不相同，但绝大多数学科的认知方式存在共同点：重视争论和证据，强调必须证明某种主张是正确的，证实某种模型能够恰如其分地反映"事物背后的道理"。

每种认知方式都兼具四种生活价值：

1. **描述**——依据特定规则、使用特定语言来描述事物，强调其特定性质和表现
2. **论证**——通过其偏好的特定争论、证据和直觉判断来论证主张、理论和观点
3. **解释**——以同样独特的方式解释其所面临的问题
4. **应用**——拥有特定的应用范围和形式

### 认知方式 vs 学科

认知方式不同于学科，但有的学科会是某一类认知方式的代表。认知方式与学科相互成就，互为原型。

## 二、前人的分类体系

### 2.1 珀金斯（Perkins）四种认知方式

出自《为未知而教，为未来而学》：

| 认知方式 | 核心特征 |
|---------|---------|
| **欧几里得式认知** | 从定义、公理和已证实的定理出发，进行形式演绎 |
| **牛顿式认知** | 强调通过数学对真实世界进行建模 |
| **培根式认知** | 侧重提出假设，再通过实验来进行验证 |
| **修昔底德式认知** | 侧重搜集、解读历史上的原始资料，批判已有解释，拼接为新的历史叙事 |

### 2.2 皮克斯通（Pickstone）五种认知方式

出自《认识方式》：

| 认知方式 | 核心特征 | 关联学科举例 |
|---------|---------|------------|
| **博物志** | 对万事万物的采集、描述、命名和分类，侧重记录多样性与变化 | 植物学、动物学 |
| **分析** | 通过解剖寻找次序，侧重分解事物，确定"已知"成分 | 分析化学、分析力学、金融分析 |
| **实验** | 将构成事物的元素组织在一起进行控制以创造新事物 | 实验心理学、实验物理、实验医学 |
| **技术科学** | 制造知识的方法，也是制造商品或武器的方法，侧重综合 | 现代商业文明 |
| **解释学** | 赋予意义与解释——无论是自然万物还是人工造物，具象还是抽象 | 宗教、科学、文学 |

### 2.3 对前人分类的评价

两位学者的共同贡献与缺陷：

- 两人都注意到了不同学科的思维方式差异，不约而同地选择了"实验"这种认知方式
- 缺陷：两人的著作都诞生于互联网时代之前，**没有看到计算机对人类认知方式的影响**
- 珀金斯的分类忽略了工程师的认知方式（即皮克斯通的"技术科学"）和文学家的认知方式（即皮克斯通的"解释学"）
- 皮克斯通没有清晰区分认知方式与学科两个概念，将不同层次的事物混在一起讨论

## 三、九种主流认知方式（核心框架）

基于前人研究，站在千年时间尺度上，阳志平将人类文明诞生以来的主流认知方式总结为九种。

### 3.1 分类总览

| 文本类型 | 认知方式 | 常见领域 |
|---------|---------|---------|
| **信息型** | 思想实验 | 像哲学家一样思考 |
| | 符号思考 | 像数学家一样思考；像统计学家一样思考 |
| | 实验科学 | 像心理学家一样思考；像生物学家一样思考；像物理学家一样思考；像化学家一样思考 |
| | 计算模拟 | 像计算机科学家一样思考；像物理学家一样思考 |
| | 田野调查 | 像人类学家一样思考；像社会学家一样思考；像历史学家一样思考；像语言学家一样思考 |
| **叙事型** | 幽默叙事 | 像段子手一样思考；像相声演员一样思考 |
| | 故事叙事 | 像小说家一样思考；像编剧一样思考 |
| **美感型** | 文采美感 | 像诗人一样思考；像散文家一样思考 |
| | 视觉美感 | 像画家一样思考；像书法家一样思考；像摄影师一样思考；像设计师一样思考 |

### 3.2 信息型认知方式

#### 3.2.1 思想实验

通过想象去进行实验，哲学家常用的认知方式。

**子类型**：
- **悖论**——通过营造矛盾放大逻辑漏洞（如忒修斯之船、沙堆悖论）
- **抽象**——使用更高级别的抽象术语体系（如美、道德、自由、意志、知识、真实、理性）
- **对话**——通过对话揭示普世真理（如《论语》《朱子语类》《传习录》《斐多》）
- **格言**——在斩钉截铁的坚定中嘲讽人性（如尼采《查拉图斯特拉如是说》、维特根斯坦《逻辑哲学论》）

**跨学科应用**：不局限于哲学，理论物理学（如爱因斯坦的"升降机""与光同行"思想实验）、理论生物学中亦常见。

**启发价值**：思想实验可启发其他认知方式。例如丹尼特的"四种心智模型"→ 埃文斯、卡尼曼的双系统理论（系统1/系统2）→ 三重心智模型（自主心智、算法心智、反省心智）。

#### 3.2.2 符号思考

使用抽象的符号而非自然语言来思考，数学家、统计学家常用的认知方式。

**子类型**：
- **定义**——清晰地列出涉及的定义、公理与定理
- **求解**——理解题目→拟订方案→执行方案→回顾（波利亚《怎样解题》）
- **维度**——独立参数的数量；认知科学中的高维空间与降维/升维操作
  - 降维示例：等距映射（Isomap）算法
  - 升维示例：支持向量机（SVM）的数学原理
- **变形**——转换事物的形态，与创造力高度相关
  - 群论：代数结构如何在变化中保持不变
  - 拓扑学：几何结构如何在变化中保持不变

#### 3.2.3 实验科学

设计各种变量，控制实验条件，求出显著差异。科学革命产生的认知方式。

**应用领域**：
- 以人为实验对象：社会学、心理学、经济学
- 以动物为实验对象：生物学、神经科学、医学
- 以自然元素为实验对象：物理学、化学、材料学

**核心洞察**：人类社会不存在复本，不得不小心翼翼地设计各种实验，推测人性、获得真知。

#### 3.2.4 计算模拟

对真实世界或虚拟世界进行抽象、建模等模拟操作，然后看事物发展趋势。计算机科学家、物理科学家常用的认知方式。

**与牛顿式认知的区别**：
- 牛顿时期：注重建模真实世界
- 21世纪（计算机联网后）：注重**规约**与**接口**

**规约**：独特的机机交互协议
- 数据库领域：三大范式规约
- 分布式计算领域：REST 规约

**接口**：
- 虚拟世界内部接口：不同计算进程交换数据（符号思考中也有）
- 连接虚拟世界与真实世界的接口（计算模拟独有）：将运算结果回传真实世界

> 符号思考与计算模拟的关键差异：符号思考在头脑中思考即可，不一定会影响真实世界；计算模拟的结果会通过各种设备使人类感知，从而影响真实世界。

#### 3.2.5 田野调查

实地参与现场的调查研究工作，人类学家、社会学家、历史学家、语言学家常用的认知方式。

**各学科的田野调查特色**：
- **人类学/社会学**：从族群、文化特例中总结规律
  - 列维-斯特劳斯《忧郁的热带》——亲访亚马孙河流域
  - 费孝通《江村经济》——江苏吴江开弦弓村调查
  - 贝特森《纳文》——新几内亚部落民族志实验
- **历史学**：不仅实地调查，还挖掘考古资料、文献（傅斯年"史学的工作就是整理史料"）
- **语言学**：寻找语言中的特例，总结规律（莱考夫《女人、火与危险事物》）

### 3.3 叙事型认知方式

#### 3.3.1 幽默叙事

通过讲述令人觉得有趣。段子手、相声演员常用的认知方式。

**三种表达形式**：
- **笑话**——最为完整的幽默叙事
- **段子**——篇目更短，保留核心意思
- **反讽**——正话反说或反话正说

**幽默的机制**：失谐-解困理论（incongruity-resolution theory）
- 只需两个事件：**营造语句**（安全情境）+ **关键语句**（出乎意料）
- 人类在安全场景中体验预期与意外的落差而发笑
- 幽默常涉及攻击或伤害（他人或自己），与温饱、安全、性爱等人类进化早期的事物有关

#### 3.3.2 故事叙事

通过讲述故事令人印象深刻。小说家、编剧常用的认知方式。

**最小故事**：任意一个故事层层剖析后的核心结构，由三个叙事事件构成：
1. **事件一（状态）**——描述行动者所处的时间、空间状态
2. **事件二（行动）**——行动者采取行动
3. **事件三（逆转）**——事件一的逆转

示例：
- 事件一：有个男人很幸福
- 事件二：他遇见一个女人
- 事件三：从此，他很不幸福

**故事性的四个原则**：
1. 事件数量为三（两个事件只能连成一条线，不足以构成立体）
2. 事件二导致事件三（人类是强因果关系的动物，先天具备脑补功能）
3. 固定的时间次序（伟大作家会像时间魔法师一样快进或放慢）
4. 逆转：事件三一定不同于事件一（人类对新奇刺激的加工强度远高于平淡事件）

### 3.4 美感型认知方式

#### 3.4.1 文采美感

通过排列组合文字、雕琢润色文字来获得美感。诗人、散文家常用的认知方式。

**核心**：唤醒"意象"

赏析示例——辛弃疾《青玉案·元夕》：
- 前七八句写群体的热闹（东风夜放花千树、宝马雕车、凤箫声动、鱼龙舞……）
- 最后一句突然转为个人的记忆（"蓦然回首，那人却在，灯火阑珊处"）
- 人人都体验过在人群中的孤独，但能如此精准表达的，唯辛弃疾

#### 3.4.2 视觉美感

通过呈现事物的整体感知来获得美感，涉及传递信息、次序和对称等。画家、书法家、摄影师、设计师常用的认知方式。

**核心**：唤醒"形象"

苏轼："少陵翰墨无形画，韩幹丹青不语诗。"

## 四、阅读四问（通用提问模板）

无论作者使用什么认知方式，都可以问自己这四类问题：

1. **描述**——本书（章节）在讨论什么事物？该事物该如何定义？它有哪些相关概念？是否包括子概念？
2. **论证**——本书（章节）支持什么？反对什么？支持的证据是什么？反对的证据是什么？作者的论证结构是什么样的？
3. **解释**——本书（章节）重点提出了什么问题？该问题能变换一种形式表达吗？作者的论证过程最终是否成功解释了该问题？是否有其他作者更好地解释了该问题？
4. **应用**——本书（章节）的结论能应用在什么领域？在该领域是否有例外？

## 五、认知方式 × 三维度分析

> 每种认知方式从「描述性、分析性、应用性」三个维度审视，对应实体作为分析入口。

| 认知方式 | 实体 | 描述性维度 | 分析性维度 | 应用性维度 |
|---------|------|-----------|-----------|-----------|
| **思想实验** | 假设 | 假设是否清晰地定义了所探讨的问题及其前提？ | 假设是否揭示了现象背后的逻辑或因果关系？ | 假设是否能够被验证或启发现实中的解决方案？ |
| **符号思考** | 符号 | 符号是否准确且明确地表达了现象的核心特征？ | 符号是否揭示了现象间的内在关联及逻辑结构？ | 符号是否能在实际问题中简化复杂性并提供洞察？ |
| **实验科学** | 实验 | 实验是否清晰描述了研究目标、设计和关键数据？ | 实验是否通过数据分析揭示了因果关系或规律？ | 实验结果是否对现实问题提供了具体可行的启发？ |
| **计算模拟** | 模型 | 模型是否清晰描述了系统的结构和条件？ | 模型是否揭示了现象背后的动态规律或逻辑机制？ | 模型能否用于预测、解释或优化现实问题？ |
| **田野调查** | 现场 | 现场是否清晰还原了关键信息？ | 调查是否揭示了现象背后的机制？ | 调查结果是否对实际问题提供了有效的行动建议？ |
| **幽默叙事** | 笑点 | 笑点是否准确呈现了现象中的矛盾或荒诞之处？ | 笑点是否揭示了背后的社会、文化或逻辑冲突？ | 笑点是否能够启发人们用新视角理解或解决问题？ |
| **故事叙事** | 故事 | 故事是否清晰描绘了情节、人物和背景？ | 故事是否揭示了现象中的深层逻辑？ | 故事是否能够引发对现实问题的反思或行动启发？ |
| **文采美感** | 意象 | 意象是否准确传达了情感或思想的核心内涵？ | 意象是否揭示了隐藏的情感深意？ | 意象是否能够激发情感共鸣或启发新的表达方式？ |
| **视觉美感** | 形象 | 形象是否清晰传达了现象的核心特征及意义？ | 形象是否揭示了现象的内在逻辑或设计原理？ | 形象是否能够启发新的思维方式或设计优化？ |

## 六、各认知方式的专门提问模板

针对不同认知方式，在通用模板的基础上可以问更具体的问题：

**思想实验**
- 作者构造了怎样的思想实验？
- 这个思想实验是作者第一次发明，还是基于历史上已有的思想实验发明的？
- 这个思想实验和你头脑中的常识有什么不同？给你提供了什么样的反常识？
- 仿照作者的思想实验，你可以用它来批判什么观点？

**符号思考**
- 作者使用的是像数学符号一样精确的符号，还是非精确、模糊的符号？
- 这些符号之间建立联系了吗？是可以运算的还是不可以运算的？
- 假如能够运算，可以得出什么结果？假如不能运算，作者用这些符号表征了什么现象？
- 模仿作者的符号可以应用到哪些领域？

**实验科学**
- 作者设计了什么类型的实验？是严格意义上的实验，还是宽松的准实验？
- 作者的实验设计是自己发明的新方法，还是业界通用的方法？
- 作者的实验假设是否存在备择假设？
- 作者的实验证据和实验结论之间证明了什么关系？是因果关系，还是相关关系，或者其他关系？
- 作者的实验是否能够脱离实验室环境，应用到更广的范围？

**计算模拟**
- 作者提出了一个什么样的计算模型？模拟了什么现象、什么事物？
- 作者的计算模型使用了真实的编程语言，还是非真实的编程语言？
- 支持作者计算模型的实际证据是什么？
- 作者的计算模型能够解释别人已有的相关研究和数据吗？

**田野调查**
- 作者是进行实地的、一手的田野调查，还是基于资料进行二手的田野调查？
- 作者进行田野调查时去了多久，待了多长时间？
- 在此之前，作者熟悉还是不熟悉调查对象？
- 作者在田野调查时得出了什么结论？这个结论有什么解释价值？能够对哪些现象进行解释？

**幽默叙事**
- 这个幽默的营造语句和关键语句是什么？
- 这个幽默与人类的什么基本需求相关？
- 这个幽默是否可笑？如果可笑，为什么可笑？如果不可笑，为什么不可笑？
- 模仿作者的幽默，可以做什么样的改写？

**故事叙事**
- 故事内核或者说最小故事是什么？这个故事的"故事性"强吗？为什么？
- 这个故事的主人翁是谁？故事发生在什么时间、地点？先后经历了哪些事件？
- 这个故事涉及人性的深层次问题吗？
- 假如要模仿作者写一个故事，你该如何写？

**文采美感**
- 作者给了你什么样的意象和感觉？
- 作者的字词、句子、篇章有哪些可以学习和模仿的地方？
- 作者唤醒了你什么样的情绪？

**视觉美感**
- 作者的作品展现了什么样的形象？
- 模仿作者的视觉设计，你可以应用到什么领域？
- 作者唤醒了你什么样的情绪？

## 七、结构阅读的冰山模型

将作者的思想看成冰山模型：

```
            ┌─────────────────┐
            │  格式规约（水面以上） │
            ├─────────────────┤
            │  方法论（水面附近）  │
            ├─────────────────┤
            │  认知方式（最底层）  │
            └─────────────────┘
```

- **认知方式**（最底层）——不同作者共享的底层。远古时代最早出现的是叙事型和美感型认知方式，随后逐渐演化出田野调查、思想实验、符号思考、实验科学、计算模拟
- **方法论**（水面附近）——学科诞生后，为解决特定问题而形成的研究方法论
- **格式规约**（水面以上）——同一学术共同体的作者形成的写作偏好规范

### 结构阅读的四个关键操作

1. **掌握作者的认知方式**——判断作者使用的主要认知方式（通常综合使用 2~4 种）
2. **向作者提问**——使用阅读四问 + 专门提问模板
3. **理解作者使用的方法论**——了解学科的研究方法论体系
4. **判断作者使用的格式规约**——识别学术共同体规范

## 八、学科研究方法论

无论人文科学、社会科学还是自然科学，都有四大典型的研究方法论：

| 方法论 | 核心特征 | 示例 |
|-------|---------|------|
| **综述研究** | 对已有文献进行搜集、整理，抽取共性和规律 | 蒋善国《尚书综述》 |
| **理论研究** | 从大量实验、现象、综述中找到规律，创建新理论 | 三重心智模型、情绪建构论、人情与面子模型 |
| **定性研究** | 深入社会现象，通过亲身体验了解研究对象，构建"情境化的"意义解释 | 拉鲁《不平等的童年》 |
| **定量研究** | 从特定假设出发，将社会现象数量化，计算相关变量之间的关系 | 津巴多《路西法效应》 |

## 十、进阶思考

### 10.1 元认知（高阶操作）

结构阅读的目的是弄清楚作者的认知方式，高阶操作就是**对认知的认知**——元认知。

**元问题**：问题的问题
- 使用"他们说过/我要说"模板，识别作者如何总结归纳他人观点
- **元评论**：评论自己的观点，告诉他人应当以及不应当如何认识这些观点（引导读者理解、消化内容）

### 10.2 系统性偏差（如何避免）

**第一类偏差**：猜错作者的认知方式
- 例如：作者使用幽默叙事，但你以为是一本正经地讨论问题
- 将符号思考等同于实验科学（如把心理学论文误解为数学）
- **解法**：平时阅读时将代表认知方式的词汇、句式不断总结下来，借助线索提高预判准确性

**第二类偏差**：不熟悉方法论和提问模板，难以提高阅读效率
- **解法**：从一两本代表作品入手反复练习，慢慢过渡到更多著作

### 10.3 何时无须了解作者的认知方式

- 多人合集
- 个人编著的教材
- 参考书

## 十一、认知方式的组合应用

- 人类大脑同时只能加工 5~9 个信息组块，最舒适的信息处理能力在 2~4 之间
- 九种认知方式可以两两组合或三三组合，穷尽出成千上万种新的可能

### 组合示例

| 组合 | 应用领域 |
|-----|---------|
| 计算模拟 + 实验科学（以人为对象） | 计算社会学、计算心理学、计量经济学 |
| 计算模拟 + 实验科学（以动物为对象） | 计算生物学、计算神经科学、计算医学 |
| 计算模拟 + 实验科学（以自然元素为对象） | 计算物理学、计算化学、计算材料学 |

### 多元认知

由多个认知方式综合在一起形成的独特认知方式，以问题为中心，常见于跨学科领域与工程技术领域。

**跨学科领域示例——认知科学**：
六个母亲学科（哲学、心理学、语言学、计算机科学、人类学、神经科学），涉及思想实验、实验科学、田野调查、计算模拟等多种认知方式。

**工程技术领域示例——软件工程**：
- 讨论需求 → 田野调查（深入用户实际场景）
- 设计原型 → 计算模拟（模拟真实世界运作规律）
- 优化算法 → 符号思考（不断求解问题，调整参数）
- 产品上线 → 实验科学（AB 测试、回归测试）
- 传播产品 → 故事叙事（开源仓库、用户故事）

## 附：推荐的认知方式配置

聪明的阅读者至少要精通四种认知方式，熟悉另外五种。精通的四种认知方式，相互距离越远越好。

> 为什么有人会给人一种书呆子的感觉？因为他只拥有单一的认知方式。

如果同时掌握 **思想实验、实验科学、计算模拟、幽默叙事** 四种距离较远的认知方式，知识结构就相对全面。反映在阅读上，应不断挑选体现不同认知方式的读物。
